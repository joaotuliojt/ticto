# Ticto

## Tecnologias

- Sass (SCSS)
- Next
- Radix
- Hook Form
- Zod

## Configuração e Execução

1. Clone este repositório:

   ```bash
   git clone https://github.com/joaotuliojt/ticto
   ```

2. Instale as dependências:

   ```bash
   cd ticto
   npm install
   ```

3. Inicie o aplicativo:

   ```bash
   npm run dev
   ```
